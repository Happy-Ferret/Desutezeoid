Desutezeoid

An XML-driven python/pygame based point and click game engine.

What Is Desutezeoid? Its an XML Driven, Point and click game engine written in python and pygame.

it features
  • sound support
  • music support
  • image support
  • an XML "page-centric" design
  • an object oriented, xml-based logic system
